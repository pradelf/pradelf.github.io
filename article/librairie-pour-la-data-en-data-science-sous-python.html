Librairie pour la data en Data Science sous Python

Pandas

Pandas est un outil d'analyse et de manipulation de données open source rapide, puissant, flexible et facile à utiliser, construit sur le langage de programmation Python.
Il s'agit de la librairie Python la plus populaire.

Dask

Dask  est un outil d'analyse et de manipulation de données open source parallèle, rapide et facile Python.

  • Dask DataFrames utilise des pandas sous le capot, donc votre code actuel fonctionne probablement.
    C'est plus rapide que Spark et plus facile aussi.
  • Parallélisez votre code Python, quelle que soit la complexité. 
    Dask est flexible et prend en charge les dépendances arbitraires et la planification des tâches à grain fin.
  • Utilisez Dask et NumPy/Xarray pour parcourir des téraoctets de données de tableau multidimensionnel dans des formats tels que HDF, NetCDF, TIFF ou Zarr.
  • Utilisez Dask avec des bibliothèques d'apprentissage automatique courantes pour vous entraîner ou prédire sur de grands ensembles de données, en augmentant la précision du modèle en utilisant toutes vos données.

 

Polar

Polars est une bibliothèque open source pour la manipulation de données, connue pour être l'une des solutions de traitement de données les plus rapides sur une seule machine. Il dispose d'une API bien structurée et typée qui est à la fois expressive et facile à utiliser.
